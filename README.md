SQL-Based Data Analysis Using Filtering, Aggregation and Business Queries

Objective
Analyze sales data using SQL with filtering, aggregation, and business queries.

Dataset
Sample Superstore Dataset

Tasks Completed
1. Loaded dataset into MySQL
2. Explored schema and sample data
3. Applied WHERE filters
4. Used GROUP BY aggregations
5. Sorted and limited results
6. Performed business analysis
7. Validated data quality

SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- SUM
- AVG
- COUNT

Key Insights
- Technology category generated high sales.
- Top customers contributed significant revenue.
- Regional sales performance varied across markets.
- Monthly sales trends showed seasonal demand patterns.
